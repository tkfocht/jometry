J!ometry

Statistics

ATT: Attempts

Number of attempts to ring in, as given in the box score.

BCOE: Buzz (Correct) Over Expectation

An estimate of how many additional response attempts a player gains through speed, or gains through opponents' lack of speed. This statistic is zero-sum: all positive BCOE must be balanced by negative BCOE for another contestant.

COB: Correct on Buzz

Correct answers on buzz. Equivalent to COR from the box score, minus any correct Daily Doubles.

CCON: Contestant Contention

An estimate of the probability that any individual correct response this contestant gave blocked an opponent from buzzing in.

DDF: Daily Doubles Found

Daily Doubles found by the player.

DDOE: Daily Doubles Over Expectation

An estimate of how many more Daily Doubles a player found versus an expectation that each correct response in a round is equally likely to lead to finding a Daily Double. This statistic is zero-sum: all positive DDOE must be balanced by negative DDOE for another contestant.

GCON: Game Contention

An estimate of the probability that any individual correct response (for any contestant) block an opponent from buzzing in.

SOB: Score on Buzz

Score using only buzz-in responses. Equivalent to actual score and to Coryat score, except it ignores Daily Doubles entirely.

This site is not affiliated with Jeopardy!. Data is taken from their published box scores.
