Statistics

ATT: Attempts

Number of attempts to ring in, as given in the box score.

BCOE: Buzz (Correct) Over Expectation

An estimate of how many additional response attempts a player gains through speed, or gains through opponents' lack of speed. This statistic is zero-sum: all positive BCOE must be balanced by negative BCOE for another contestant.

COB: Correct on Buzz

Correct answers on buzz. Equivalent to COR from the box score, minus any correct Daily Doubles.

CCON: Contestant Contention

An estimate of the probability that any individual correct response this contestant gave blocked an opponent from buzzing in.

DDF: Daily Doubles Found

Daily Doubles found by the player.

DDOE: Daily Doubles Over Expectation

An estimate of how many more Daily Doubles a player found versus an expectation that each correct response in a round is equally likely to lead to finding a Daily Double. This statistic is zero-sum: all positive DDOE must be balanced by negative DDOE for another contestant.

GCON: Game Contention

An estimate of the probability that any individual correct response (for any contestant) block an opponent from buzzing in.

SOB: Score on Buzz

Score using only buzz-in responses. Equivalent to actual score and to Coryat score, except it ignores Daily Doubles entirely.

Contention Metrics

For the purposes of calculating contention metrics, all incorrect responses and all Daily Doubles are ignored. This is because incorrect responses do not prevent others from attempting to buzz in after the response, and because Daily Doubles are not contended on the buzzers at all. It is true that not all attempts to buzz in that are blocked by an incorrect response are reattempted (the incorrect response may be the response the blocked contestant intended to give, or they may not wish to follow up for other reasons), but in an attempt to quantify and estimate a level of competition, this is ignored.

The base data are then Correct on Buzz, the number of correct responses on buzz-in clues, and Blocked Attempts, the number of Attempts minus Correct on Buzz minus Incorrect on Buzz. As an example, in S38R Game 111, the base data are:

Contestant J! COB J! Blocked DJ! COB DJ! Blocked
Dave Rapp 13 5 13 4
Carrie Christian 8 11 5 4
Hung Pham 2 15 4 6

Each contestant is credited with 1 attempt for each COB they actually made, because we know for sure this attempt occurred. The blocked attempts for each contestant are allocated equally across their opponent's COB. So Dave is credited with 5/10 of an attempt for each COB by Carrie or Hung, Carrie is credited with 11/15 of an attempt for each COB by Dave or Hung, and Hung is credited with 15/21 for each COB by Dave or Carrie. This allocation ignores the possibilities of blocked attempts that are not followed up on Triple Stumpers, and it ignores the rare instances of double correct responses.

With this in mind, we can estimate CCON and GCON. CCON is the probability that any one of a contestant's COB responses blocked at least one opponent's attempt. For Dave, in the J! Round, this is:

CCON = (11/15) + (15/21) - ((11/15)*(15/21)) = 0.92 = 92%

Note that because of the assumptions around Triple Stumpers, the raw CCON calculation can go above 100%. In these cases, it is capped at 100%.

GCON is the weighted average of the contestants' CCONs, which for the J! Round is:

GCON = (13(Dave's CCON) + 8(Carrie's CCON) + 2(Hung's CCON))/(13+8+2) = 0.90 = 90%

With the allocations in hand, we can also estimate an expected number of buzzes a contestant should have had, continuing to ignore existing incorrect responses and the possibility of further incorrect responses. For each of his own buzzes, we estimate an expected value for Dave of:

1 / (1 + (11/15) + (15/21)) = 0.409 buzzes

But additionally, he has an expected value of buzzing for each of Carrie's buzzes of:

(5/10) / ((5/10) + 1 + (15/21)) = 0.226 buzzes

And for Hung's of:

(5/10) / ((5/10) + (11/15) + 1) = 0.224 buzzes

Weighted and combined, this is:

13(0.409) + 8(0.226) + 2(0.224) = 7.57 buzzes total

Since Dave actually had 13 COB, his BCOE is 13 - 7.57 = +5.43 buzzes.
